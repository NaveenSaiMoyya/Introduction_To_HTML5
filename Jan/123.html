IMAGE Tag

=> inline Element.
=> supports many image file types.
                    > JPEF
                    > JPG
                    > GIF
                    > PNG
                    > SVG
                    > BMP

=> file extensions must be included.
=> every image must be downloaded 
=> every image requires an HTTP request.

Image Attributes: 
                - width, height, alt

* alt must be always included as it is read by screen readers and helps in SEO.


FAVICON

=> img/logo/icon next to the title of page.
=> must be in <head> section.
=> can't use JPEG or GIF for logos.
