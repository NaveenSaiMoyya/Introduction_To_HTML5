HTML-5:
    - cooperation between W3C & WHATWG (Web Hypertext Application Technology Working Group)

HTML5 Guidelines:
    - New features should only based on HTML, CSS, DOM & Javascript.
    - reduce need for external plug-ins
    - more markup to replace scripting
    - device independent

  
      *v* whenever designers want to do something new they write non-standard code to force 
      browsers to do it.

      *v* New Standards are written to handle new requirements &
      browsers must adopt new standards.
