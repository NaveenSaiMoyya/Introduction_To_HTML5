 HTML - HyperText Markup Language 
     - markup files so that browsers know how to display content on webpage
     - Tags are used to distinguish between content. 

 WebPage - not just one file, but many pieces (different types of files) put together
          by browser using Request/Response Cycle.

 Syntax of Tags: How to use

 Semantics of Tags: When to use (Meaning of tags)

 Accessible Pages: accessed by widest range of people all over the world.
                          (including physical & cognitive disabilities)
                 -  by using special accessibility tools (Screen readers)

 ** All webPages are built upon DOM (document object model)
