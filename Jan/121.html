ATTRIBUTES


=> Provides additoinal info about an element.
=> always specified in start tag.
=> Name/Value pairs.


Global Attributes


=> applies for all tags.

class attribute: grouping of elements.
id attribute: unique id for an element.
style attribute: inline css (not recommended)
accesskey: shortcut key to activate an element.
tabindex: order elements coming into focus using tab key.
