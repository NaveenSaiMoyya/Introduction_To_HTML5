aria-label attribute:
          - used by screen readers to read out the text.

Anchor Links:
          - <a> is used for hyperlinks.
          - href attribute -> hyper reference
          - content between start and end tag is displayed.

Types of links:
ABSOLUTE: external link to another webpage.
RELATIVE: same folder file.
INTERNAL (EMBEDDED): jumps in same page.
GRAPHICAL or IMAGE LINKS: image as a link.


target attribute for LINKS:
                    - self
                    - blank
                    - top
                    - parent
