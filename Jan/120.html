TAGS


=> Every tag has a beginning and an end.
=> Some are self closing tags which contain both befinning and ednding.
=> Tags have attributes.

-_-_-_ Display Attribute -_-_-_

- most crucial attribute of an element: Display 
Display values: 
                - Block
                - inline
                - Inline-Block

Block: - newline is inserted before and after 
       - takes up whole width 
       - headings, paragraph, divisions

Inline: take as much space as needed for an eleemnt.

=> Some tags are inherently block and some are inherently Inline.
